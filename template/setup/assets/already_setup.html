Setup Revisited

Template Blocks thinks you have run this setup before. Possible situations may be:

 

Updating...

You have just uploaded a newer version of Template Blocks and want to start using it. In this case click the following button.

 

Interrupted Installation...

You have interrupted a previous installation and Template Blocks is still not working. In this case, please click the following button to override this warning and re-initiate the setup process.

 

Setup folder not removed...

You have successfully completed the setup process but you are redirected here from the administration page because you still have the setup folder online. Please remove the "setup/" folder and all of it's contents before someone takes advantage of it and something really bad happens ;)

 

If there is something else going wrong please visit the Template Blocks support pages and request for help.
