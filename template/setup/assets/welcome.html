Initiating Installer

Welcome to the setup process of Template Blocks. We promise this will not take much of your time.

Before anything else let's outline the requirements to be able to work with Template Blocks.

  • Your domain is pointing to a server where you have read-write permissions
  • Your server has PHP 5 installed
  • You have access to a MySQL database, and
  • You are using a browser with JavaScript and CSS enabled

Although these technologies are fairly common these days some people may have problem using any one of them. All of the above must be available for the recipe to work ;)

To make things even easier for you this installer has been created with the same ingredients as the administration interface so if you can use the setup you won't have any problem using Template Blocks itself.
